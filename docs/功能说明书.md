# Inner Cosmos 功能说明书

## 用户系统

支持注册、登录、退出、当前用户 Session 校验，提供默认 `demo` 和 `admin` 账号。

## Aurora 今日对话

用户可选择今日倾诉、思维整理、睡前复盘、苏格拉底追问四种模式。系统支持文字输入、Mock 语音转写、Aurora 朋友式回应和对话结束后的整理流程。

## 今日记录卡

对话结束后生成结构化记录，包括今日主题、情绪天气、认知碎片、待办线索和 Aurora 总结。

## 记忆星空

MemoryCard 会计算情感重力，并以星体形式展示。用户可点击星体查看摘要，并进入共鸣体编织流程。

## 思维碎纸机

用户可一次性输入混乱文本，系统提取可沉淀的认知卡片，而不是直接公开或长期暴露原始文本。

## 共鸣体与星海广场

用户可基于脱敏记忆创建 EchoCapsule。星海广场展示用户授权共鸣体和系统种子共鸣体，支持有限轮次对话。

## 慢信件

用户可写慢信，信件经过 DRAFT、SENT、FLYING、DELIVERED、READ、REPLIED、DECLINED、BLOCKED、ARCHIVED 等状态流转。

## 安全边界

系统对高风险词、骚扰、人肉等内容进行拦截，并引导到安全资源页。产品不做心理诊断，不替代医生、咨询师或热线。

## 管理后台

后台展示用户、共鸣体、举报、模型配置提示和运营概览，默认不展示用户 P0 原始私密内容。
