# Inner Cosmos 内宇宙 — 课程大作业提交说明

> **基于 Java Web 与大模型 API 的 AI 自我共鸣与慢社交平台**
> 一个以 AI 伴侣「Aurora」为核心的「自我记录 + 情感重力记忆 + 授权共鸣体 + 慢信件」AI Agent 系统。
> 它**不是**「输入框 → 调用大模型 → 显示回答」的普通聊天网站。

---

## 一、五分钟运行（老师无需安装 Maven）

默认使用 **H2 文件数据库 + 真实大模型 GLM（glm-4.7）**，**无需 MySQL，也无需配置任何 API Key 或环境变量**——真实 API key 已内置在提交的源码里，开箱即真实 AI。

1. 确认本机已安装 **JDK 17 或更高版本**（命令行执行 `java -version` 可查看）。
2. 进入 `可执行程序/` 目录：
   - **Windows：双击 `run-jar.bat`**；或
   - 命令行执行：`java -jar inner-cosmos-0.1.0.jar`
3. 启动约需 10–20 秒，浏览器打开：**<http://localhost:8080/pages/index.html>**
4. 登录账号：

   | 账号 | 密码 | 角色 |
   | --- | --- | --- |
   | `demo` | `demo123` | 普通用户（推荐体验，已预置丰满演示数据） |
   | `admin` | `admin123` | 管理员（含管理后台） |

> **默认即接入真实大模型 GLM glm-4.7（智谱旗舰，key 已内置，请求里已关闭 deep-thinking 以保证约 6–8 秒响应）**，老师一登录即可看到真实 AI 对话、主动问候、多维画像、Aurora 自我模型等全部特性。失败转移链为 **GLM → MiniMax → Mock 兜底**：若网络/额度异常，系统自动回退，演示不中断。天气（open-meteo）与时间（浏览器本地）本就是真实数据、无需 key。如需切换供应商或自带 key，可用环境变量 `LLM_PROVIDER` / `LLM_API_KEY` / `GLM_API_KEY` 等覆盖（见 `文档/安装说明书.md` 的"远程模型配置"）。

## 四、技术栈

- **后端**：Java 17、Spring Boot 3.3.6、Spring Security、MyBatis-Plus、Bucket4j 限流、JWT、Actuator + Micrometer/Prometheus
- **数据库**：H2 文件库（默认，零配置）/ MySQL 8（可选，Profile 切换）
- **AI**：默认真实 GLM glm-4.7（key 内置、已关 deep-thinking 调优响应）+ 自研 PromptBuilder 多阶段装配 + StructuredAiService 结构化抽取 + 多供应商失败转移（GLM → MiniMax）+ 关键词感知 Mock 兜底
- **前端**：原生 HTML/CSS/JS（无构建步骤），SSE 流式对话，深色「内宇宙」美学，完整可访问性（skip-link / focus-trap / aria-live / reduced-motion）
