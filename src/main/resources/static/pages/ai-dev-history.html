关键工程决策

Mock-first AI 架构

课程环境无稳定 API Key，所有 AI 调用默认走 Mock 模板，配置真实 Key 后自动切换真实模型。Mock 内容经过精心撰写，不是占位符。

MVC 严格分层

Controller 只做参数校验和路由，Service 包含全部业务逻辑，Mapper 继承 MyBatis-Plus BaseMapper。VO/DTO 隔离内外数据结构。

安全边界设计

AI 是镜子不是医生。SafetyBoundaryFilter 在模型调用前拦截；共鸣体是授权回声不是真人替代；慢信延迟机制防止即时情绪冲动。

情感重力算法

MemoryCard 的 emotionalGravity = intensity * recurrence^0.7 * importance * (1 + triggers*0.1)，综合情绪强度、重复次数、用户评分、触发次数。不是简单计数。

H2 演示 + MySQL 生产

默认 H2 内存数据库零配置启动，application.yml 中提供 MySQL 配置模板，切换只需改 spring.datasource.url。

设计理念

不是要让用户「变好」，而是提供一个安全的空间，让混乱的感受可以先被看见、被命名、被整理。

暖雾 Morandi 色系、缓慢的信件传递、非即时回复 — 每一个设计选择都在降低情绪压力。
