设计语言说明

  • 白天（dawn / morning / noon / afternoon）：莫兰迪色系（暖米 / 灰绿 / 抹茶 / 暮橙）
  • 夜晚（night / deep-night）：温暖星云（夜褐 / 烛光 / 月光银），不用纯黑不用冷蓝
  • 动效曲线：ease-bloom 入场（缓启动）、ease-settle 离场（快走慢停）
  • 流动母题：背景四层（基础渐变 + 粒子 + 纹理 + 水墨晕染）永远在动
  • 用户感知：从不弹通知，时间和天气是潜移默化的渐变
